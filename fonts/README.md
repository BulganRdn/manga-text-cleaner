# Fonts

Drop `.ttf` / `.otf` font files into this folder and they appear in the
Studio's font picker (press the refresh button in the picker, or restart the
app). Fonts installed in the operating system are picked up automatically —
this folder is for fonts you want to ship with the project without
installing them system-wide.

Make sure you have the right to use a font in published pages; many
scanlation-style fonts are free for non-commercial use only.
